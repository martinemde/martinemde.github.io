# Welcome to the Self-Rendering Markdown Page This page was written as raw markdown and enhanced *entirely client-side* with a tiny script. ## Why? - Easy to write - Friendly for LLMs and tools - No build step required - Still looks nice in a browser ## What's Happening? 1. You open this file or request it from a server 2. The raw markdown is rendered by your browser 3. A script at the bottom kicks in and: - Reads the markdown - Parses it into HTML - Replaces the content with styled HTML ---
